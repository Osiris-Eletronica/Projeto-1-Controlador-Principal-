Dei início ao projeto para ajudar alunos a entender a mecânica, a elétrica e a lógica de programação.
Mas procurando conscientizar esses alunos da segurança devida quando se trata do elevador.
O elevador é um aparelho super seguro, mas, ao mesmo tempo, se o técnico não tiver consciência, ele se torna de extrema periculosidade.
O elevador é tão perigoso para quem está fazendo manutenção ou consertando, seu risco de vida é eminente e pode deixar até aleijado?
